# Medical Expenses Linear Regression

This project demonstrates the implementation of linear regression using Scikit-Learn in Python to predict medical expenses. The notebook covers:

- Downloading and exploring the dataset
- Implementing linear regression with one and multiple variables
- Utilizing categorical features
- Analyzing regression coefficients and feature importance

## Files

- `Med Expenses_Linear Regression.ipynb`: Jupyter Notebook containing the code and explanations.
- `README.md`: This guide explaining the project and its files.
- 'Notebook PDF': PDF of Jupyter Notebook.

## Dataset

The dataset used in this project can be found [here](https://github.com/stedy/Machine-Learning-with-R-datasets/blob/master/insurance.csv).

## Note on Interactive Graphs 

Due to GitHub's static rendering of Jupyter Notebooks, interactive Plotly graphs do not display directly. For full visualization, please refer to the included PDF file of the notebook, which contains all plots.
